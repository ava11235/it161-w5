zyBooks Ch 4: Basic CSS

CSS, Cascading Style Sheets, associates style rules with HTML elements.

CSS treats each HTML element as if it appears inside its own box. CSS applies rules to indicate the look of the element.

Rules are made up of selectors, which specify which elements the rule applies to, and declarations, which say what these elements should be styled like.

Different types of selectors allow for targeting the rules at different elements.

Declarations are made up of two parts: the properties of the element to be styled, and the values of those properties.

CSS rules usually appear in their own document, although they can also be added within the HTML page
